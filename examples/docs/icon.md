## Icon
基于 [Font Awesome](http://www.fontawesome.com.cn/faicons/) 提供了一套常用的图标集合。

:::demo 基于 Font Awesome 封装, type 属性传递 icon 名称即可。
```html
  <template>
    <ki-icon style='margin-right: 20px;' type='check' />
    <ki-icon style='margin-right: 20px;' type='close' />
    <ki-icon type='info' />
  </template>

  <script>
    export default {
      setup() {},
    };
  </script>
```
:::
